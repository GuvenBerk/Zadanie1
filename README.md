# Menadżer Zadań z Autoryzacją

Aplikacja do zarządzania zadaniami z systemem rejestracji i logowania.

## Funkcjonalności

- Rejestracja i logowanie użytkowników
- Dodawanie, edycja i usuwanie zadań
- Ustawianie priorytetów i terminów zadań
- Publiczna strona główna
- Bezpieczne hashowanie haseł
- Autoryzacja JWT
